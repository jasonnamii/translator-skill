# 번역 QC 체크리스트 (상세)

번역 완료 후 **필수 실행**. 이 체크리스트를 건너뛰면 구조 편차가 그대로 전달된다.

**QC 통합:** 자동 검증 스크립트 1차 실행. 스크립트 통과 항목은 수동 QC 스킵. 스크립트 실패/경고 항목만 수동 QC 8항 중 해당 항목 확인.

| # | 검증 항목 | 방법 | 실패 사례 |
|---|----------|------|----------|
| 1 | **헤딩 수 일치** | `##`/`###` 카운트 = 원본 카운트 | 번역 중 서브섹션 누락 |
| 2 | **콜아웃 수 일치** | `[!type]` 카운트 = 원본 카운트 | ID에서 4개 콜아웃 누락 |
| 3 | **div 스타일 일치** | `<div style="...">` 속성값 전수 비교 | border·padding·radius 값 임의 변경 |
| 4 | **위키링크 유효성** | TOC `[[#heading]]` = 실제 헤딩 텍스트 | 헤딩 번역 후 TOC 동기화 누락 |
| 5 | **고유명사·카피 보존** | 원본의 고유명사·영어 카피가 번역본에서 변형 없이 존재 | 브랜드명 현지어 번역 |
| 6 | **파일 크기 편차** | 바이트 크기 ±25% 이상 → 내용 편차 의심 | TH +40% (UTF-8 멀티바이트 정상이나 확인) |
| 7 | **내용 추가/삭제 없음** | 번역본에 원본에 없는 섹션·콜아웃 추가 금지 | ID에 원본에 없는 신규 섹션 추가 |
| 8 | **격식체 일관성** | 언어별 지정된 격식체 기준과 일치 | 경어/반말 혼용 |

## QC 실행 방법

### 1. 자동 검증 — `qc_diff.py` (필수)

```bash
# 테이블 출력 (사람 보고용)
python3 ${CLAUDE_SKILL_DIR}/scripts/qc_diff.py <원본.md> <번역본.md>

# JSON 출력
python3 ${CLAUDE_SKILL_DIR}/scripts/qc_diff.py <원본.md> <번역본.md> --json
```

### 자동 검증 커버리지 (13항목)

| 기본 검증 (카운트) | 확장 검증 (difflib/위치 특정) | 퍼지 검증 (의미적) |
|-------------------|--------------------------|---------------------|
| 헤딩 수 (H1-H6) | 헤딩 시퀀스 정렬 — 누락·추가·레벨 불일치 위치 | — |
| 콜아웃 수 | — | — |
| 테이블 수 | — | — |
| div 개수 | div style 속성값 전수 비교 | — |
| 코드블록 수 | — | — |
| 위키링크 수 | TOC `[[#heading]]` ↔ 실제 헤딩 교차 검증 | — |
| 이미지 임베드 수 | — | — |
| 리스트 아이템 수 (±10% 허용) | — | — |
| 파일 크기 비율 (±50% 경고) | — | — |
| — | — | **문단 대응** — 타입별 블록 수 비교 |
| — | — | **미번역 잔류 감지** — 8자↑ 연속 + 40%↑ 비율 |

- exit code 0 = 전항목 PASS, 1 = FAIL.
- `--json` 모드: `{"passed": bool, "summary": {...}, "extended_issues": [...]}`.
- **FAIL 1건이라도 = 수정 후 재실행.** PASS 전까지 번역 완료 선언 금지.

### 2. 수동 검증
격식체, 고유명사, 톤 → 서두 + 중간 + 말미 샘플링.

### 3. 편차 발견 시
원본 기준으로 수정. 번역본 독자 판단에 의한 구조 변경 불허.
